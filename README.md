# ShapleySlides2018
Slides and Proceedings article from 2018 Sawtooth Software Conference paper by David W. Lyon, "Shapley Values: Easy, Useful and Intuitive".

The article is taken directly from the Proceedings of the 2018 Sawtooth Software Conference.  The slides are a slightly updated version of those presented at the conference.

The software referenced near the end of the paper is no longer at the URL given.  Instead, see github.com/DWLyon17/ShapCOA for updated and more comprehensive software for Shapley Values and TURF analyses (among other things).

The paper includes mention of sampling of combinations as a mechanism for approximation of Shapley Values when problems are very large.  The author no longer recommends this; see github.com/DWLyon17/COAslides26 for a paper and slides on a better approach, samplign orderings using Component Order of Addition (COA) designs.

This paper remains useful partly as an introduction to SCs, but mostly for its details on the "hypergeometric trick" that allows easy, fast and exact computation of Shapley Values for 0-1 TURF problems and many variants of 0-1 TURF.
