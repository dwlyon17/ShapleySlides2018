# ShapleySlides2018
Slides and Proceedings article from 2018 Sawtooth Software Conference paper by David W. Lyon, "Shapley Values: Easy, Useful and Intuitive".

The article is taken directly from the Proceedings of the 2018 Sawtooth Software Conference.  The slides are a slightly updated version of those presented at the conference.

The paper and slides discuss the idea of taking random samples of combinations as the basis for approximate computation of the Shapley Value.  That is no longer recommended by the author.  Instead, using COA (Component Order of Addition) designs as a way to sample different orderings is recommended, as both easier to implement and faster, for more accuracy.  See the COAslides2026 and ShapCOA repositories under github.com/DWLyon17 for more details.
